# TFG-LLM
Notebooks asociados al TFG de Manuel García González - "Una primera aproximación teórico-práctica a los Grandes Modelos de Lenguaje (LLMs)"

En ellos podemos hallar una breve comparación de las capacidades de los modelos DistilBERT y BERT, ajustados con un dataset concreto y realizando una tarae de clasificación de texto.
